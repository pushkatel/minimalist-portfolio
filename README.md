# Kush Patel — Personal Portfolio

A minimalist personal portfolio site built with plain HTML and CSS, hosted on Firebase.

## Tech Stack

- HTML5 / CSS3
- Google Fonts (Roboto)
- Font Awesome icons
- Firebase Hosting

## Project Structure

```
public/
  index.html   # Single-page portfolio
  style.css    # Styles and responsive layout
firebase.json  # Firebase Hosting config
```

## Running Locally

Open `public/index.html` in a browser — no build step required.

## Deploying

```bash
firebase deploy
```
